## ■ Technical Learnings

**1. Rails: redirect_toメソッドとrenderメソッドの処理フローの違い**

**renderメソッド**
- 新しいHTTPリクエストを行わずに処理を行う
- 同じコントローラーの別のviewファイルを返すことでリクエスト回数を減らしている

**redirect_toメソッド**
- 新しくHTTPリクエストが発生する

**2. Rails: フラッシュメッセージの基本的な使い方**

- Q：flashメソッドを使うためにはMVCのどこに記述する必要があるのか？
- A：コントローラーとビューファイルとセットで作成

**Controller**

```ruby
if article.save
  redirect_to path, flash: {キー名: "表示させたいメッセージ"}
else
  # 略
end
```

**View**

フラッシュメッセージを表示させたい箇所に下記を記述。コントローラーで定義した[キー名]をhtmlファイルに記述することでメッセージ部分を表示させることができる

`<%= flash[:キー名] %>`

```erb
例) html.erb
<%= flash[:alert] %>
```

**Evidence**:
- https://zenn.dev/take_tech/articles/62468f3f149e5b#render-%E3%81%A8%E3%81%AF%E4%BD%95%E3%81%8B%EF%BC%9F

**3. CSS: Tailwind CSS 背景色・テキスト色の変更**

背景：`bg-{色}-{濃さ}` の形式で指定

```html
<!-- 例 -->
<div class="bg-white">白い背景</div>
<div class="bg-red-100">薄い赤の背景</div>
```

テキスト：`text-{色}-{濃さ}` の形式で指定

```html
<!-- 例 -->
<div class="text-gray-800">グレーのテキスト</div>
<div class="text-green-700">緑のテキスト</div>
```

**Evidence**:
- https://toach.biz/tools/tailwind-css-cheat-sheet/?view=colorpicker
- https://zenn.dev/yohei_watanabe/books/c0b573713734b9/viewer/215e30
