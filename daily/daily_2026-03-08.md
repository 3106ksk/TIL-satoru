## ■ Technical Learnings

**1. Rails: localsオプション**
- **Question**: localsオプションとは?
- **Answer**: renderメソッドのオプションとして使用。`locals: {キー名: コレクション}` と使うことでローカル変数としてパーシャルファイルに渡すことができる

**2. Rails: コレクションレンダリングと省略記法が使える条件**
- **Question**: なぜ以下のように省略記法が使えるのか?
  ```erb
  <% @products.each do |product| %>
    <%= render partial: "product", locals: { product: product } %>
  <% end %>
  ```
  省略記法:
  ```erb
  <%= render partial: "product", collection: @products %>
  ```
- **Answer**: パーシャルファイル名、ローカル変数、インスタンス変数名が一緒の場合に省略記法が規約によって使うことができる
- **Evidence**:
  - https://railsguides.jp/v7.0/action_view_overview.html#%E3%83%91%E3%83%BC%E3%82%B7%E3%83%A3%E3%83%AB%E3%83%AC%E3%82%A4%E3%82%A2%E3%82%A6%E3%83%88
  - https://qiita.com/gon0821/items/a314b16ff04a9e460add

**3. daisyUI: ビューでの使用方法**
- **Question**: daisyUIでリストもしくはカードを作成する方法
- **Answer**: 公式よりデザインを参照。HTMLのタグにBootstrapの時と同様にCSSクラスを割り当てていく

**4. Rails: アセットパイプライン**
- **Question**: Rails環境においてのアセットパイプラインとは
- **Answer**: JS、CSS、画像ファイルなどの静的アセットを整理・キャッシュ・配信するために設計されたライブラリのこと

**5. Tailwind CSS/daisyUI: 環境構築手順**
- **Answer**:
  ②Gemfileに以下を追記してインストールを実行
  ```ruby
  gem "jsbundling-rails"
  gem "cssbundling-rails"
  ```
  `docker-compose exec web bundle install` をターミナルで入力し、bundle install
  ③ `docker-compose exec web bundle exec rails css:install:tailwind` を実行
  ④yarnでdaisyUIをインストール
  `docker-compose exec web yarn add daisyui` をターミナルで入力して実行
  ⑤tailwind.config.jsの作成と編集
- **Evidence**:
  - https://qiita.com/ICU1234/items/4c7c3e974ae290e68792
  - https://zenn.dev/yoiyoicho/articles/410a7e3fd892b5
  - https://v3.tailwindcss.com/docs/content-configuration
