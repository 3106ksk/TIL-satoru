■ Technical Learnings

**1. Rails: コントローラーとビューのスコープ**
- **Question**: 該当のユーザーを見つけるときは`current_user`だけを使えばよいか？`show`アクションでビューに渡す変数が必要では？
- **Hypothesis**: `show`定義時に`@user`などが必要だと思った。
- **Conclusion**: ビュー内で直接`current_user`ヘルパーメソッドを使用しているため、コントローラーのアクションでのインスタンス変数定義は不要だった。
- **Code**:
  ```ruby
  class ProfilesController < ApplicationController
    def show; end
  end
  ```

**2. Rails: helper_method**
- **Question**: コントローラーで定義しているメソッド（`current_user`等）がなぜビューでも使用できるのか？
- **Conclusion**: `helper_method`を使用しているため。これによりコントローラーメソッドをビューヘルパーとして公開できる。
- **Evidence**: [AbstractController::Helpers::ClassMethods](https://api.rubyonrails.org/classes/AbstractController/Helpers/ClassMethods.html)
- **Code**:
  ```ruby
  class ApplicationController < ActionController::Base
    helper_method :logged_in?, :current_user
    # ...
  end
  ```
