## ■ Technical Learnings

**1. Ruby: ハッシュ引数の省略記法**
- **Question**: `User.exists?(reset_password_token:)` の引数のシンボルについてなぜこの書き方なのか？
- **Answer**: メソッドの引数にハッシュを使うときの省略記法。{}の省略 + key/valueが同名の場合のvalue省略が可能（Ruby 3.1+）
- **Code**:
  ```ruby
  User.exists?(reset_password_token: reset_password_token)  # フル記述
  User.exists?({ reset_password_token: reset_password_token })  # ハッシュとして明示
  User.exists?(reset_password_token:)                       # 省略記法
  ```

**2. Rails: ActiveRecordのカラムアクセス**
- **Question**: `self.reset_password_token = reset_password_token` はメソッド呼び出しかカラム代入か？
- **Conclusion**: ActiveRecordはDBカラムに対して自動的にgetter/setterメソッドを生成。カラム名でメソッドとしてアクセス可能。
- **Reference**: https://zenn.dev/wwwave/articles/2e80f16aac8974
- **Code**:
  ```ruby
  content = Content.find(21)
  content.price  # 自動生成されたメソッドを通じてカラム値にアクセス
  ```
