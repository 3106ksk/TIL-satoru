1) Cognitive Headroom（思考の余白）1–5 定義

何を測る？
「考える余裕が残っていたか」。眠気というより、脳内の“スペース”。
5：ひらけてる（余白たっぷり）迷ってもすぐ戻れる。複数の手を並行で捌ける
言語化・整理が自然にできる
例：詰まっても「じゃあ仮説立てよう」が即出る

4：十分ある
基本スムーズ。判断が遅れない
30〜60分の集中が作りやすい

3：ふつう（波あり）
できるけど、立ち上がりに時間がいる
少し複雑になると頭が詰まりやすい

2：狭い（余裕少なめ）
思考が重い。判断が遅い。小さな選択でも疲れる
途中で“休憩を挟まないと崩れる”

1：ほぼ無い（カツカツ）
何をするにも苦しい。読む/考えるが進まない
“作業を回すだけ”になりがち

✅10秒チェック（Headroom）
「今この状態で、別の問題が来ても対処できる？」
YES → 4-5 / まあまあ → 3 / NO → 1-2

2) Focus（注意の維持）1–5 定義
何を測る？
「注意がどれだけ1点に留まれたか」。成果じゃなく“注意の安定”。
5：吸い込まれた（ほぼ途切れない）
気づいたら時間が進んでいる
中断ゼロ〜1回、復帰が速い

4：かなり安定
小さな揺れはあっても、すぐ戻れる
25分単位で集中が保てる

3：並（集中と散漫が半々）
たびたび意識が逸れるが、タスクは続けられる
立ち上がりに10分くらいかかることが多い

2：不安定（すぐ逸れる）
何度も他へ飛びたくなる/飛ぶ
「戻る」コストが高い

1：ほぼ集中できない
セッションの大半が途切れている
何をしてたか見失う

✅10秒チェック（Focus）
「最後の20分、注意はどれだけ作業に張り付いてた？」
ほぼずっと → 5 / 大体 → 4 / 半分 → 3 / 少し → 2 / ほぼ無理 → 1

3) Friction（引っかかり度）1–5 定義
何を測る？
「進むまでの摩擦」。迷い・詰まり・割り込み・道具の不便、全部込み。
※高いほど“ザラつき”。

1：スルスル（摩擦ほぼ無し）
次の一手が明確、作業が滑る
中断があっても復帰が即

2：少し引っかかる
小さな詰まりはあるが、すぐ解消

3：普通（摩擦そこそこ）
調べ物/迷い/中断が何回かある
“前に進む感”はあるがテンポは落ちる

4：高い（何度も止まる）
同じところで詰まる/調べ物が深追い/割り込み多い
「進んだ時間」より「戻す時間」が増える

5：非常に高い（ぐるぐる/進めない）
error_loop / research_loop に入って抜けにくい
ほぼ前進しない、または強いストレス

✅10秒チェック（Friction）
「前に進むのに、余計な抵抗がどれだけあった？」
ほぼ無し→1 / 少し→2 / 普通→3 / 多い→4 / ほぼそれ→5

迷いやすいパターンの“裁定”ルール（超重要）
この3つは 同時に高低がズレてOKです。むしろズレが鏡になります。
Focus高い × Friction高い：
→ “歯を食いしばって集中（error_loopで粘った等）”
Headroom低い × Focus高い：
→ “短距離は刺さるが消耗が激しい”
Headroom高い × Focus低い：
→ “余裕はあるのに散ってる（通知/環境/目的不明）”

入力をさらに楽にする「アンカー（基準点）」おすすめ

毎回同じ基準でつけられるよう、あなた用にこの3つだけ覚えると楽です。

5：理想（再現したい状態）

3：普通（可もなく不可もなく）

1：崩壊（次の一手が必要）

あとは「2 or 4」は 1段階寄せるだけ。

Notionに貼れる短い定義（プロパティ説明用）

Cognitive Headroom：考える余裕（1=カツカツ / 3=普通 / 5=ひらけてる）
Focus：注意の安定（1=ほぼ逸れる / 3=半々 / 5=没入）
Friction：進む抵抗（1=スルスル / 3=普通 / 5=ぐるぐる）
